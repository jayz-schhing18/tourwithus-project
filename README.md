🌊 TourWithUs
Explore the Heart of Koh Rong – Cambodia’s Island Paradise.

TourWithUs is a specialized web platform designed to showcase the beauty of Koh Rong. From luxury sea-base resorts to hidden local gems, we help travelers discover the perfect stay, breathtaking views, and must-visit spots across the island.

✨ Features
Curated Accommodations: A list of the best hotels categorized by type (Resorts, Bungalows, Eco-stays).

Panoramic Views: High-quality showcases of the white sand beaches and turquoise sea-base views.

Destination Guide: Highlights of the best places to visit around the Koh Rong archipelago.

Interactive Design: A sleek, responsive interface built for travelers.

🏝️ Island Highlights
Hotel Types: Over-water villas, beachfront huts, and luxury suites.

Top Locations: Koh Touch, Long Set Beach (4K Beach), and Saracen Bay.

Activities: Bio-luminescent plankton tours, diving, and island hopping.

🚀 Tech Stack
Frontend: HTML5, CSS3 (Tailwind CSS)

Logic: JavaScript

Version Control: Git & GitHub
